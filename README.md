# Frontend Mentor – QR Code Component

This project is part of the **FreeDev Pair Challenge Program**, where mentees work on real-world frontend challenges under the guidance of a mentor.

The goal of this project is to recreate the QR Code Component design using **Vanilla HTML and CSS** while following professional development practices such as GitHub workflow, issue tracking, and code reviews.

---

## Challenge

This project is based on the Frontend Mentor challenge:

https://www.frontendmentor.io/challenges/qr-code-component-iux_sIO_H

Frontend Mentor challenges help developers improve their coding skills by building realistic projects.
